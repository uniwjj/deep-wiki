CDC жҳҜд»Җд№Ҳ

CDC жҳҜеҸҳжӣҙж•°жҚ®жҚ•иҺ·пјҲChange Data CaptureпјүжҠҖжңҜзҡ„зј©еҶҷпјҢе®ғеҸҜд»Ҙе°Ҷжәҗж•°жҚ®еә“пјҲSourceпјүзҡ„еўһйҮҸеҸҳеҠЁи®°еҪ•пјҢеҗҢжӯҘеҲ°дёҖдёӘжҲ–еӨҡдёӘж•°жҚ®зӣ®зҡ„пјҲSinkпјүгҖӮеңЁеҗҢжӯҘиҝҮзЁӢдёӯпјҢиҝҳеҸҜд»ҘеҜ№ж•°жҚ®иҝӣиЎҢдёҖе®ҡзҡ„еӨ„зҗҶпјҢдҫӢеҰӮеҲҶз»„пјҲGROUP BYпјүгҖҒеӨҡиЎЁзҡ„е…іиҒ”пјҲJOINпјүзӯүгҖӮ

дҫӢеҰӮеҜ№дәҺз”өе•Ҷе№іеҸ°пјҢз”ЁжҲ·зҡ„и®ўеҚ•дјҡе®һж—¶еҶҷе…ҘеҲ°жҹҗдёӘжәҗж•°жҚ®еә“пјӣA йғЁй—ЁйңҖиҰҒе°ҶжҜҸеҲҶй’ҹзҡ„е®һж—¶ж•°жҚ®з®ҖеҚ•иҒҡеҗҲеӨ„зҗҶеҗҺдҝқеӯҳеҲ° Redis дёӯд»ҘдҫӣжҹҘиҜўпјҢB йғЁй—ЁйңҖиҰҒе°ҶеҪ“еӨ©зҡ„ж•°жҚ®жҡӮеӯҳеҲ° Elasticsearch дёҖд»ҪжқҘеҒҡжҠҘиЎЁеұ•зӨәпјҢC йғЁй—Ёд№ҹйңҖиҰҒдёҖд»Ҫж•°жҚ®еҲ° ClickHouse еҒҡе®һж—¶ж•°д»“гҖӮйҡҸзқҖж—¶й—ҙзҡ„жҺЁз§»пјҢеҗҺз»ӯ D йғЁй—ЁгҖҒE йғЁй—Ёд№ҹдјҡжңүж•°жҚ®еҲҶжһҗзҡ„йңҖжұӮпјҢиҝҷз§ҚеңәжҷҜдёӢпјҢдј з»ҹзҡ„жӢ·иҙқеҲҶеҸ‘еӨҡдёӘеүҜжң¬ж–№жі•еҫҲдёҚзҒөжҙ»пјҢиҖҢ CDC еҸҜд»Ҙе®һзҺ°дёҖд»ҪеҸҳеҠЁи®°еҪ•пјҢе®һж—¶еӨ„зҗҶе№¶жҠ•йҖ’еҲ°еӨҡдёӘзӣ®зҡ„ең°гҖӮ

дёӢеӣҫжҳҜдёҖдёӘзӨәдҫӢпјҢйҖҡиҝҮи…ҫи®Ҝдә‘ Oceanus жҸҗдҫӣзҡ„ Flink CDC еј•ж“ҺпјҢеҸҜд»Ҙе°ҶжҹҗдёӘ MySQL зҡ„ж•°жҚ®еә“иЎЁзҡ„еҸҳеҠЁи®°еҪ•пјҢе®һж—¶еҗҢжӯҘеҲ°дёӢжёёзҡ„ RedisгҖҒElasticsearchгҖҒClickHouse зӯүеӨҡдёӘжҺҘж”¶з«ҜгҖӮиҝҷж ·еӨ§е®¶еҸҜд»Ҙеҗ„иҮӘеҲҶжһҗиҮӘе·ұзҡ„ж•°жҚ®йӣҶпјҢдә’дёҚеҪұе“ҚпјҢеҗҢж—¶еҸҲе’ҢдёҠжёёж•°жҚ®дҝқжҢҒе®һж—¶зҡ„еҗҢжӯҘгҖӮ

CDC зҡ„е®һзҺ°еҺҹзҗҶ

йҖҡеёёжқҘи®ІпјҢCDC еҲҶдёәдё»еҠЁжҹҘиҜўе’ҢдәӢд»¶жҺҘж”¶дёӨз§ҚжҠҖжңҜе®һзҺ°жЁЎејҸгҖӮ

еҜ№дәҺдё»еҠЁжҹҘиҜўиҖҢиЁҖпјҢз”ЁжҲ·йҖҡеёёдјҡеңЁж•°жҚ®жәҗиЎЁзҡ„жҹҗдёӘеӯ—ж®өдёӯпјҢдҝқеӯҳдёҠж¬Ўжӣҙж–°зҡ„ж—¶й—ҙжҲіжҲ–зүҲжң¬еҸ·зӯүдҝЎжҒҜпјҢз„¶еҗҺдёӢжёёйҖҡиҝҮдёҚж–ӯзҡ„жҹҘиҜўе’ҢдёҺдёҠж¬Ўзҡ„и®°еҪ•еҒҡеҜ№жҜ”пјҢжқҘзЎ®е®ҡж•°жҚ®жҳҜеҗҰжңүеҸҳеҠЁпјҢжҳҜеҗҰйңҖиҰҒеҗҢжӯҘгҖӮиҝҷз§Қж–№ејҸдјҳзӮ№жҳҜдёҚж¶үеҸҠж•°жҚ®еә“еә•еұӮзү№жҖ§пјҢе®һзҺ°жҜ”иҫғйҖҡз”ЁпјӣзјәзӮ№жҳҜиҰҒеҜ№дёҡеҠЎиЎЁеҒҡж”№йҖ пјҢдё”е®һж—¶жҖ§дёҚй«ҳпјҢдёҚиғҪзЎ®дҝқи·ҹиёӘеҲ°жүҖжңүзҡ„еҸҳжӣҙи®°еҪ•пјҢдё”жҢҒз»ӯзҡ„йў‘з№ҒжҹҘиҜўеҜ№ж•°жҚ®еә“зҡ„еҺӢеҠӣиҫғеӨ§гҖӮ

дәӢд»¶жҺҘж”¶жЁЎејҸеҸҜд»ҘйҖҡиҝҮи§ҰеҸ‘еҷЁпјҲTriggerпјүжҲ–иҖ…ж—Ҙеҝ—пјҲдҫӢеҰӮ Transaction logгҖҒBinary logгҖҒWrite-ahead log зӯүпјүжқҘе®һзҺ°гҖӮеҪ“ж•°жҚ®жәҗиЎЁеҸ‘з”ҹеҸҳеҠЁж—¶пјҢдјҡйҖҡиҝҮйҷ„еҠ еңЁиЎЁдёҠзҡ„и§ҰеҸ‘еҷЁжҲ–иҖ… binlog зӯүйҖ”еҫ„пјҢе°Ҷж“ҚдҪңи®°еҪ•дёӢжқҘгҖӮдёӢжёёеҸҜд»ҘйҖҡиҝҮж•°жҚ®еә“еә•еұӮзҡ„еҚҸи®®пјҢи®ўйҳ…е№¶ж¶Ҳиҙ№иҝҷдәӣдәӢд»¶пјҢз„¶еҗҺеҜ№ж•°жҚ®еә“еҸҳеҠЁи®°еҪ•еҒҡйҮҚж”ҫпјҢд»ҺиҖҢе®һзҺ°еҗҢжӯҘгҖӮиҝҷз§Қж–№ејҸзҡ„дјҳзӮ№жҳҜе®һж—¶жҖ§й«ҳпјҢеҸҜд»ҘзІҫзЎ®жҚ•жҚүдёҠжёёзҡ„еҗ„з§ҚеҸҳеҠЁпјӣзјәзӮ№жҳҜйғЁзҪІж•°жҚ®еә“зҡ„дәӢд»¶жҺҘж”¶е’Ңи§ЈжһҗеҷЁпјҲдҫӢеҰӮ DebeziumгҖҒCanal зӯүпјүпјҢжңүдёҖе®ҡзҡ„еӯҰд№ е’Ңиҝҗз»ҙжҲҗжң¬пјҢеҜ№дёҖдәӣеҶ·й—Ёзҡ„ж•°жҚ®еә“ж”ҜжҢҒдёҚеӨҹгҖӮ

з»јеҗҲжқҘзңӢпјҢдәӢд»¶жҺҘж”¶жЁЎејҸж•ҙдҪ“еңЁе®һж—¶жҖ§гҖҒеҗһеҗҗйҮҸж–№йқўеҚ дјҳпјҢеҰӮжһңж•°жҚ®жәҗжҳҜ MySQLгҖҒPostgreSQLгҖҒMongoDB зӯүеёёи§Ғзҡ„ж•°жҚ®еә“е®һзҺ°пјҢе»әи®®дҪҝз”Ё Debezium жқҘе®һзҺ°еҸҳжӣҙж•°жҚ®зҡ„жҚ•иҺ·пјҲдёӢеӣҫжқҘиҮӘ Debezium е®ҳж–№ж–ҮжЎЈпјүгҖӮеҰӮжһңдҪҝз”Ёзҡ„еҸӘжңү MySQLпјҢеҲҷиҝҳеҸҜд»Ҙз”Ё CanalгҖӮ

дёәд»Җд№ҲйҖү Flink

д»ҺдёҠеӣҫеҸҜд»ҘзңӢеҲ°пјҢDebezium е®ҳж–№жһ¶жһ„еӣҫдёӯпјҢжҳҜйҖҡиҝҮ Kafka Streams зӣҙжҺҘе®һзҺ°зҡ„ CDC еҠҹиғҪгҖӮиҖҢжҲ‘д»¬иҝҷйҮҢжӣҙе»әи®®дҪҝз”Ё Flink CDC жЁЎеқ—пјҢеӣ дёә Flink зӣёеҜ№ Kafka Streams иҖҢиЁҖпјҢжңүеҰӮдёӢдјҳеҠҝпјҡ

  • Flink зҡ„з®—еӯҗе’Ң SQL жЁЎеқ—жӣҙдёәжҲҗзҶҹе’Ңжҳ“з”Ё
  • Flink дҪңдёҡеҸҜд»ҘйҖҡиҝҮи°ғж•ҙз®—еӯҗе№¶иЎҢеәҰзҡ„ж–№ејҸпјҢиҪ»жқҫжү©еұ•еӨ„зҗҶиғҪеҠӣ
  • Flink ж”ҜжҢҒй«ҳзә§зҡ„зҠ¶жҖҒеҗҺз«ҜпјҲState BackendsпјүпјҢе…Ғи®ёеӯҳеҸ–жө·йҮҸзҡ„зҠ¶жҖҒж•°жҚ®
  • Flink жҸҗдҫӣжӣҙеӨҡзҡ„ Source е’Ң Sink зӯүз”ҹжҖҒж”ҜжҢҒ
  • Flink жңүжӣҙеӨ§зҡ„з”ЁжҲ·еҹәж•°е’Ңжҙ»и·ғзҡ„ж”ҜжҢҒзӨҫзҫӨпјҢй—®йўҳжӣҙе®№жҳ“и§ЈеҶі
  • Flink зҡ„ејҖжәҗеҚҸи®®е…Ғи®ёдә‘еҺӮе•ҶиҝӣиЎҢе…Ёжүҳз®Ўзҡ„ж·ұеәҰе®ҡеҲ¶пјҢиҖҢ Kafka Streams еҸӘиғҪиҮӘиЎҢйғЁзҪІе’Ңиҝҗз»ҙ

иҖҢдё” Flink Table / SQL жЁЎеқ—е°Ҷж•°жҚ®еә“иЎЁе’ҢеҸҳеҠЁи®°еҪ•жөҒпјҲдҫӢеҰӮ CDC зҡ„ж•°жҚ®жөҒпјүзңӢеҒҡжҳҜеҗҢдёҖдәӢзү©зҡ„дёӨйқўпјҢеӣ жӯӨеҶ…йғЁжҸҗдҫӣзҡ„ Upsert ж¶ҲжҒҜз»“жһ„пјҲ+I иЎЁзӨәж–°еўһгҖҒ-U иЎЁзӨәи®°еҪ•жӣҙж–°еүҚзҡ„еҖјгҖҒ+U иЎЁзӨәи®°еҪ•жӣҙж–°еҗҺзҡ„еҖјпјҢ-D иЎЁзӨәеҲ йҷӨпјүеҸҜд»ҘдёҺ Debezium зӯүз”ҹжҲҗзҡ„еҸҳеҠЁи®°еҪ•дёҖдёҖеҜ№еә”гҖӮ

Flink CDC зҡ„дҪҝз”Ёж–№жі•

зӣ®еүҚ Flink CDC ж”ҜжҢҒдёӨз§Қж•°жҚ®жәҗиҫ“е…Ҙж–№ејҸгҖӮ

иҫ“е…Ҙ Debezium зӯүж•°жҚ®жөҒиҝӣиЎҢеҗҢжӯҘ

дҫӢеҰӮ MySQL -> Debezium -> Kafka -> Flink -> PostgreSQLгҖӮйҖӮз”ЁдәҺе·Із»ҸйғЁзҪІеҘҪдәҶ DebeziumпјҢеёҢжңӣжҡӮеӯҳдёҖйғЁеҲҶж•°жҚ®еҲ° Kafka дёӯд»ҘдҫӣеӨҡж¬Ўж¶Ҳиҙ№пјҢеҸӘйңҖиҰҒ Flink и§Јжһҗе№¶еҲҶеҸ‘еҲ°дёӢжёёзҡ„еңәжҷҜгҖӮ

еңЁиҜҘеңәжҷҜдёӢпјҢз”ұдәҺ CDC еҸҳжӣҙи®°еҪ•дјҡжҡӮеӯҳеҲ° Kafka дёҖж®өж—¶й—ҙпјҢеӣ жӯӨеҸҜд»ҘеңЁиҝҷжңҹй—ҙд»»ж„ҸеҗҜеҠЁ/йҮҚеҗҜ Flink дҪңдёҡиҝӣиЎҢж¶Ҳиҙ№пјӣд№ҹеҸҜд»ҘйғЁзҪІеӨҡдёӘ Flink дҪңдёҡеҜ№иҝҷдәӣж•°жҚ®еҗҢж—¶еӨ„зҗҶе№¶еҶҷеҲ°дёҚеҗҢзҡ„ж•°жҚ®зӣ®зҡ„пјҲSinkпјүеә“иЎЁдёӯпјҢе®һзҺ°дәҶ Source еҸҳеҠЁдёҺ Sink зҡ„и§ЈиҖҰгҖӮ

з”Ёжі•зӨәдҫӢ

еүҚж–Үи®ІиҝҮпјҢFlink CDC Connector еңЁеҲқе§Ӣзҡ„е…ЁйҮҸеҝ«з…§еҗҢжӯҘйҳ¶ж®өпјҢдјҡеұҸи”ҪжҺүеҝ«з…§зҡ„жү§иЎҢпјҢеӣ жӯӨеҰӮжһң Flink Checkpoint йңҖиҰҒжү§иЎҢзҡ„иҜқпјҢе°ұдјҡеӣ дёәдёҖзӣҙж— жі•иҺ·еҫ— checkpointLock еҜ№иұЎзҡ„й”ҒиҖҢи¶…ж—¶гҖӮ

ж—§зүҲиҜӯжі•зҡ„ Connector еңЁ JDBC жү№йҮҸеҶҷе…Ҙ Upsert ж•°жҚ®пјҲдҫӢеҰӮж•°жҚ®еә“зҡ„жӣҙж–°и®°еҪ•пјүж—¶пјҢе№¶жңӘиҖғиҷ‘еҲ° Upsert дёҺ Delete ж¶ҲжҒҜд№Ӣй—ҙзҡ„йЎәеәҸе…ізі»пјҢеӣ жӯӨдјҡеҮәзҺ°й”ҷд№ұзҡ„й—®йўҳпјҢиҜ·е°Ҫеҝ«иҝҒз§»еҲ°ж–°зүҲзҡ„ Flink SQL иҜӯжі•гҖӮ
