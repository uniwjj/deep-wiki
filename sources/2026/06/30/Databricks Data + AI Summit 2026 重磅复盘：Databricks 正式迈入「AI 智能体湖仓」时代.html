二、年度最大变革：从湖仓一体 → 智能体湖仓一体

过去几年，行业认知是：Delta Lake + 湖仓一体 = 统一数据底座。

而 2026 DAIS 给出全新标准答案：

旧架构：Delta Lake + 湖仓一体 → 数据能存、能算，但不会主动干活
新架构：Lakebase（在线事务）+ Delta/Iceberg（离线湖仓）+ Genie/Agent（AI 智能层）= 企业 Agent 操作系统

全新智能体湖仓架构：数据 + 计算 + AI 三位一体

这是本届峰会最底层、最颠覆性的架构升级，彻底解决行业多年痛点：

  • 传统数据平台：数据能存、能算，但不会主动干活
  • 早期 Agent：能对话、能调用工具，但缺统一数据、缺治理、无法生产落地

全新架构实现：企业所有数据、所有任务、所有分析，全部由 AI 智能体自主驱动、自主运维、自主迭代。

三、七大重磅新品全面发布（全是 GA 可落地）

1. Agent Bricks — 企业生产级 AI 智能体平台（正式 GA）

本届真正的王牌产品，企业 Agent 工业化落地标准方案。告别「Demo 级智能体」，直接实现生产可用：

  • 多智能体 Supervisor 调度：自动拆解复杂业务任务、串联多工具、分步执行、自动纠错
  • 智能体全自动评测体系：自动生成测试数据、评测幻觉、准确率、成本，持续迭代优化
  • 统一模型网关：一键接入 OpenAI / Anthropic / 开源大模型，智能路由、故障降级、成本管控
  • 全链路安全治理：深度绑定 Unity Catalog，权限、审计、血缘、行列级隔离全覆盖
  • 内置文档智能解析：非结构化文件自动抽取、入库、构建企业私有知识库

价值一句话：让企业 AI 智能体从「玩具」变成「可规模化落地的业务系统」。

2. Genie One 全新升级 — 全员数据智能统一入口

Databricks 正式统一品牌，Genie 成为唯一数据 AI 入口，分两大角色覆盖全员：

Genie（业务人员）

  • 自然语言查数、自动生成报表、智能可视化
  • 自动学习企业业务术语，大幅降低问答幻觉
  • 支持移动端、Teams/Slack 嵌入、外部免账号访问

Genie Code（技术人员 · 全页模式 Beta）

  • 自动写 SQL / Notebook / 数据流水线 / 模型脚本
  • 7×24 小时监控任务，自动修复管道故障、模型衰退
  • 多线程并行开发，大幅提升研发效率

3. Lake TAP + Reyden 双引擎 — 底层架构重构

Lake TAP：打通离线海量湖数据 + 在线实时业务数据，无需数据同步、无需多库跳转，智能体可毫秒级混合读写全域数据。

Reyden 全新交互式查询引擎：替代旧版交互计算，向量 + 全文 + SQL 混合查询毫秒响应，专门为 AI 问答、智能体调用、实时分析提速。

4. Lakebase 全域 GA — 三大云全量上线

AI 智能体专属原生 Postgres 数据库，正式全云可用（AWS / Azure / GCP）。

  • 湖仓统一元数据、统一权限、统一审计
  • 弹性扩缩、Scale-to-zero、分支开发、秒级恢复
  • 完美适配：RAG 在线存储、智能体业务操作、低延迟 API、实时业务系统

从此，Databricks 真正实现：一套平台搞定离线分析 + 在线业务 + AI 智能服务。

Lakebase：离线 + 在线 + AI 智能服务统一底座

5. AI Search — 原向量检索全面升级（正式 GA）

告别单一向量检索，行业首个三合一混合检索引擎：

  • 向量检索（RAG 知识库）
  • 全文检索（传统文本搜索）
  • 结构化过滤（数据表精准筛选）

支持 Delta / Iceberg 表一键建索引、自动增量同步，企业私有化 RAG 部署门槛直接降到最低。

6. Unity Catalog 重大突破 — 原生 Iceberg 写入 GA

此前仅支持 Iceberg 读，本届正式开放完整读写能力！

  • Delta + Iceberg 双格式原生统一治理
  • 一套 Catalog 管理两种主流湖仓格式
  • 跨引擎无拷贝共享数据，彻底弱化厂商锁定

这是 Databricks 面向开放中立数据生态最重要的一次升级。

7. Unity AI Gateway + 收购 Panther Labs — AI 安全治理闭环

  • Unity AI Gateway：
    统一管控所有 LLM 调用，实现脱敏、权限、预算、成本分摊、审计全管控
  • 收购 Panther Labs：
    补齐 AI 行为威胁检测、异常审计、数据防泄露能力

至此，Databricks 完成 AI 生产、调度、落地、安全、治理全栈闭环。
