## 核心主题（基于关键词推断）

### Token 冗余问题
- **冗余越大** — LLM 对话中存在大量 Token 浪费，包括重复上下文、无效轮次、过长 prompt

### 消耗性连续与流量膨胀
- **消耗性连续** — Token 消费是连续的、累积的过程，单次对话可能产生大量消耗
- **流量膨胀** — 随着 AI 应用规模扩大，Token 消费量呈膨胀式增长

### 预算治理
- **预算治理** — 需要建立 Token 消费的预算和控制机制
- **完整治理链路** — 从预算分配、实时监控到告警限流的完整治理体系

### 可控放量
- **实现可控放量** — 在保障业务需求的前提下，实现 Token 使用的可控增长

### 敏感输出
- **敏感输出** — Token 消费中可能涉及敏感信息的输出，需要特别关注和治理
