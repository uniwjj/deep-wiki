# Prompt 工程

Prompt 工程是设计和优化 LLM 输入提示词的方法论，直接影响模型输出的质量和可控性。

## 核心技术

- **结构化 Prompt**：角色设定、任务描述、输出格式约束
- **Few-shot Prompting**：通过示例引导模型行为
- **Chain-of-Thought**：引导模型逐步推理
- **Prompt 模板化**：可复用、可组合的 prompt 组件
