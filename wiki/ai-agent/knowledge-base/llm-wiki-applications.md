# LLM Wiki 应用场景

## 核心类比

> "传统 RAG 是'每次都从原材料做饭'，LLM Wiki 是'LLM 帮你建立私人厨房，提前备好半成品'。"

## 五大应用场景

| 场景 | 做法 | 价值 |
|------|------|------|
| **个人第二大脑** | 整合日记/笔记/播客/健康数据 | 构建个人成长图谱 |
| **专题研究** | 论文/博客/报告纳入，LLM整理核心概念 | 梳理研究脉络、标注矛盾 |
| **深度阅读** | 每章内容纳入，自动生成人物关系/主题脉络 | 关键情节关联 |
| **团队协作** | 整合 Slack/会议/文档 | 知识不随人员流动流失 |
| **专项任务** | 竞品分析、旅行规划等 | 动态方案生成 |

## 四步上手

1. **准备工具**：Obsidian + Claude Code/Codex + 原始资源
2. **对接 Agent**：复制 Karpathy Gist 内容给 Agent
3. **Agent 搭建**：创建目录结构→schema→INDEX.md+log.md
4. **启动积累**：拖入资源(Agent自动处理)→查询知识→定期Lint

## 核心结论

LLM Wiki 让 AI 成为全职知识管理员，人类聚焦深度思考与创意产出。
