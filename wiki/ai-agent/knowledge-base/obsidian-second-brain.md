# Obsidian 个人第二大脑

> "它从来不是一款'死板的工具'，而是一个'可定制的知识管理平台'。"

## 四大核心优势

### 1. 本地存储，数据自主
- 纯文本 Markdown 格式存本地，不依赖任何云端服务器
- 可在文件夹中直接增删查改，VSCode/Everything 等软件联动
- 数据流动完全自由，不被任何平台绑定

### 2. 超高自定义
- 官方本体功能 + 核心插件（可自由开关）
- 近 **1000 个社区插件** + 160 多款主题 + 自定义 CSS 片段
- 覆盖编辑体验优化、工作流自动化、本地软件联动、在线服务对接

### 3. 双向链接
- 输入 `笔记名` 创建链接，不存在则点击自动创建
- 右侧面板显示反向链接和提及，查看所有引用关系
- 关系图谱插件可视化笔记关联网络

### 4. 活跃社区生态
- 全球社区提供大量免费教程和实践分享
- Zotero 文献管理、Pandoc 格式转换等高阶玩法均有成熟教程
- 个人使用完全免费，支持 Windows/macOS/Linux

## 推荐插件栈

| 插件 | 功能 |
|------|------|
| **Templater + Calendar** | 一键生成日记模板，降低日记录入门槛 |
| **Weread** | 微信读书笔记一键同步到 Obsidian |
| **Dataview** | 搭建自动更新仪表板，一次配置永久生效 |
| **Iconic** | 给文件/文件夹设置专属图标 |
| **豆包集成** | 笔记内直接调用 AI 整理格式 |

## 多设备同步

搭配 OneDrive 实现免费多设备同步：PC 端深度创作、移动端同步查阅。

## 快速上手

1. 下载安装（全平台免费）→ 创建 Vault（本地文件夹）→ 新建笔记
2. `Ctrl/Cmd + P` 命令面板快速触发功能
3. 核心插件在「设置-核心插件」中自由开关
4. 社区插件安装：设置→关闭安全模式→浏览→搜索→安装→启用（全程 < 1 分钟）
